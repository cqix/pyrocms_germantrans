Multilingual Howto

Follow this howto to add another database to PyroCMS to support different languages.

  1. Copy the existing database and give it a rememberable name.

    Example

    If your main database is called pyrocms and you want to add another database for the German version call it pyrocms_german.

  2. Open up system/codeigniter/pyrocms/config/database.php and add the configuration for your second language.

    Name the new configuration entry live_ followed by the 2 character language code.

    Example

    $db['live_de'] = $db['live'];
$db['live_de'] = 'pyrocms_german';

  3. You might want to add a language switch on the frontend.

    Use the Theme "Minimal with language support" to do so.
